cnum

Comfortably Numbers

Comfortably Numbers (cnum) represents rational numbers and related mathamatical expressions in a biginteresting way.

Install

 
yarn add cnum

Example Usage

 
import Rat from 'cnum/Rat'

const a = new Rat(71, 7) // a = 71/7
const b = new Rat(5, 791) // b = 5/791
const c = a.multiply(b) // c = a*b

c.toString() // 355/113
c.valueOf() // 3.14159292035
