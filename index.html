cnum - Comfortably Numbers

Comfortably Numbers (cnum) represents rational numbers and related mathamatical expressions in a biginteresting way.

Install

 
yarn add cnum

Usage

Build System

 
import {Rat} from 'cnum'

// a = 71/7
const a = new Rat(71, 7)

// b = 35/113
const b = new Rat(35, 113)

// c = aâ‹…b
const c = a.mul(b)

c.toString() // 355/113
c.valueOf() // 3.1415929203539825

Node

 
const {Rat} = require('cnum')

const r = new Rat(7, 11)

r.continuedFraction()

Script Tags

 
<script src="//unpkg.com/cnum"></script>

const r = new Rat(4, 13)

for (let n of r.continuedFraction()) {
  console.log(n)
}
